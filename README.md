# 🐄 Pashu-Aahar

Pashu-Aahar is a Smart Cattle Nutrition Calculator Android application developed using Kotlin and Android Studio.  
The app helps farmers generate optimized cattle feed plans, calculate nutrition values, track feed history, and estimate feed cost savings.
